# chatbot-demo
Intent Classifier Chatbot for Hospitality industry in 2 languages:

* Persian
* English

Persian version is based on 5 intention classes:

* Navigation ([data](https://docs.google.com/spreadsheets/d/1JHUFjzoL6oHmYBpFFyJMh2qtqNP7VdRH6qwBtLcEbjI/edit?usp=sharing))
* Food Orders ([data](https://docs.google.com/spreadsheets/d/1ICXZYX52l9V69xY4t_DbpUoqvAURjnCGqL60Nsj1Oik/edit?usp=sharing))
* Room Service ([data](https://docs.google.com/spreadsheets/d/1yHRWF8bo36q2opeLuv-yLJ_hmOrP77tiAevjf2gUSPg/edit?usp=sharing))
* Dry Cleaning Orders ([data](https://docs.google.com/spreadsheets/d/1Wvt9Nx7XTya5q_qVU6hEYfnNv-G5_YU39DEvbtiPXMI/edit?usp=sharing))
* Weather ([data](https://docs.google.com/spreadsheets/d/17H2tehXX02jmdfcV8f58eEAz7lRsb2N7Y3km0J7SQW8/edit?usp=sharing))

We created our own dataset for persian version that led to outstanding results in testing. We also used [Fasttext](https://fasttext.cc/docs/en/english-vectors.html) persian word vectors that helped us a lot.
